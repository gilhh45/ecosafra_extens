Arquitetura backend (Django)
ecosafra_hub/
├── config/settings/
│   ├── base.py       → SECRET_KEY, LANGUAGE_CODE, TIME_ZONE (America/Porto_Velho)
│   ├── local.py      → SQLite + DEBUG=True
│   └── production.py → PostgreSQL + HTTPS + HSTS + CSP
└── apps/
    ├── clients/      → Model: Client (CPF único, IntegrityError em duplicata)
    ├── credentials/  → EncryptedTextField AES-256 (cryptography lib)
    └── documents/    → ServiceOrder (SEC-YYYY-PVH-NNNN) + Document (S3)
Para produção:
1. PostgreSQL + select_for_update() — evita race conditions na geração de IDs
2. EncryptedTextField AES-256 — senhas criptografadas no banco
3. django-storages + AWS S3 — upload de PDFs em nuvem com URL privada
4. Gunicorn + Nginx + HTTPS — acesso remoto seguro
5. pywebview — encapsula como app desktop nativo
